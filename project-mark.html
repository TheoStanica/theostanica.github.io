About

Project mark is a Social Media Manager that aims to help companies better manage their social media accounts. Mark allows users to connect multiple Twitter accounts, tweet messages for multiple accounts at the same time and customize the dashboard streams that follow specific terms. This platform is my first big project that I am developing. To develop this application I have used multiple technologies such as Node, MongoDB, Redis, Node Nats Streaming, Docker, Kubernetes, React and Redux as well as Twitter API.
