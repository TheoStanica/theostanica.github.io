About

GitTixApp is an Android application created to consume the GitTix API. Together, they allow users to buy and sell event tickets. Payments are integrated using Stripe. GitTix API is written in Typescript and has a microservice architecture. It is deployed using Docker and Kubernetes. GitTix also has a minimal React web application. Technologies used: Typescript, React, Android Studio, MongoDB, Redis, Node-nats-streaming, Stripe.
